## Tech Stack:
- RxJava
- Retrofit2
- Room
- Glide
- Koin 
- Timber
- Mockito

## How to use it ?
- Clone this project using `https://github.com/gusentanan/moviee.git`
- Open project using the latest version of Android Studio.
- Run the project.
- Run unit tests using `./gradlew clean test`
